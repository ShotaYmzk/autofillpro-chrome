# AutoFillPro（Chrome 拡張）

就活・採用エントリーなどのマイページで、名前・連絡先・住所・学歴などを **ワンクリックで自動入力** する Chrome 拡張機能です。プロフィールは端末内だけに保存され、複数プロフィールの切り替えや郵便番号からの住所補完に対応しています。

**バージョン:** manifest の `version` を参照（例: `chrome-extension/manifest.json`）

## できること

- **複数プロフィール**（メイン／デモ用など）の作成・切り替え・保存
- **ツールバーのポップアップ**から現在のタブへ自動入力
- **対応サイト限定**で、ページ右下の **「自動入力」フローティングボタン**（設定で変更可）
- **学歴・学校区分**（大学／大学院など）に応じた表示や、Axol などサイト側ラジオとの整合
- **帰省先＝現住所と同じ**のチェック時は、設定画面上で帰省先入力欄を折りたたみ表示
- **郵便番号検索**（ZipCloud API）による住所補完（オプション画面・設定の動作に準拠）
- 設定の **エクスポート／インポート**（JSON）

## インストール（開発版の読み込み）

1. このリポジトリをクローンまたは ZIP で入手します。
2. Chrome で `chrome://extensions` を開きます。![オプション画面](/Users/yamazakiakirafutoshi/VScode/Self_intro/docs/screenshots/readme-chrome-extensions-load-unpacked.png)
3. 右上の **デベロッパーモード** をオンにします。
![オプション画面２](/Users/yamazakiakirafutoshi/VScode/Self_intro/docs/screenshots/readme-toolbar-extensions-menu.png)
4. **パッケージ化されていない拡張機能を読み込む** を選び、フォルダ `**chrome-extension`** を指定します。

ストア公開の `.crx` ではなく、上記はローカル読み込み（アンパック）の手順です。

## 画面つきガイド（どこを押すか）

### A. 初回だけ：`chrome-extension` フォルダを読み込む

まず `**chrome://extensions**` で、次のボタンからフォルダを選びます。

デベロッパーモードをオンにしたうえで「パッケージ化されていない拡張機能を読み込む」をクリック


| 手順  | 操作                                                 |
| --- | -------------------------------------------------- |
| 1   | アドレスバーに `chrome://extensions` と入力して開く              |
| 2   | 右上の **デベロッパーモード** を **オン** にする                     |
| 3   | **パッケージ化されていない拡張機能を読み込む** をクリック（↑のボタンと同じ）          |
| 4   | ファイル選択で、このリポジトリ内の `**chrome-extension`** フォルダを指定する |


### B. いつも：ツールバーの「拡張機能」から AutoFillPro を開く

入力したいページを開いた状態で、ポップアップから自動入力できます。

右上のパズルアイコン → 一覧の AutoFillPro をクリック


| 手順  | 操作                                                          |
| --- | ----------------------------------------------------------- |
| 1   | ブラウザ右上の **パズルピースのアイコン（拡張機能メニュー）** をクリック                     |
| 2   | 一覧から **AutoFillPro - 就活マイページ自動入力** をクリック → **ポップアップ** が開きます |
| 3   | （おすすめ）同じ一覧の **ピン留め** をオンにすると、アイコンが常時表示されて次から開きやすくなります       |


オプション画面でプロフィールを編集するときは、ポップアップ右上の **設定（歯車アイコン）** をクリックするか、ツールバーの拡張機能アイコンを **右クリック** → **オプション** から開けます。

## 使い方

### 1. 初回セットアップ（オプション画面）

1. 拡張機能のアイコンを右クリック → **オプション**（または拡張機能の詳細からオプションを開く）。
2. **基本情報・連絡先・学歴** などを入力します。
3. 画面上部の **保存する** で `chrome.storage.local` に保存されます。
  - 注意文どおり、**保存するまで**自動入力側は最新の未保存内容を読みません。

### 2. プロフィール

- 左サイドバーでプロフィールを切り替えられます。
- **未保存のまま別プロフィールに切り替える**と確認ダイアログが出ます。

### 3. 自動入力の実行

- **ポップアップ**: 拡張アイコンをクリック → 「自動入力」など（ポップアップの UI に従う）。  
→ **どのサイトでも** 試せます（汎用マッチング）。
- **フローティングボタン**: 設定が有効かつ条件を満たすページの右下に緑ボタンが出ます。  
→ 既定では **Axol / i-Web / Entry Sheet など専用アダプタが効くページのみ**（後述）。

## 対応サイト・入力の仕組み


| 種類              | 説明                                                                |
| --------------- | ----------------------------------------------------------------- |
| **サイト専用アダプタ**   | `axol.jp`、`i-web` / Hunet 系、エントリーシート検出ページなど。フィールド名やフローに合わせて入力します。 |
| **汎用（Generic）** | ラベルや name 属性などから推測して入力します。                                        |


フローティングボタンの表示は、詳細設定の **「すべてのサイトでフローティングボタンを表示」** で制御できます（オフ＝専用アダプタが選ばれるページにのみ表示／オン＝任意サイトでも表示されうる旧挙動に近い）。

## オプション（詳細設定）


| 項目                    | 概要                              |
| --------------------- | ------------------------------- |
| 入力済みフィールドをハイライト       | 入力した場所を視覚的に強調                   |
| 入力前にプレビュー             | 実行前にモーダルで確認                     |
| ページ内に「自動入力」ボタンを表示     | フローティングボタンのオン／オフ                |
| すべてのサイトでフローティングボタンを表示 | **オフ推奨**。オンにすると無関係なサイトにもボタンが出ます |
| 入力間隔（ms）              | フィールド間の待機。サイトが追従しないときは増やす       |
