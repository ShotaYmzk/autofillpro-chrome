# AutoFillPro プライバシーポリシー

最終更新日: 2026年5月14日

本ドキュメントは、Chrome 拡張機能「AutoFillPro - 就活マイページ自動入力」（以下「本拡張機能」）における、情報の取り扱いについて説明します。

## 1. 開発者・問い合わせ

本拡張機能はオープンソースとして公開されています。ご質問やご指摘は、リポジトリの Issues などからご連絡ください。

- リポジトリ: https://github.com/ShotaYmzk/autofillpro-chrome

（運営主体・連絡先メールを追加する場合は、上記に追記してください。）

## 2. 収集する情報と保存場所

**本拡張機能は、開発者が運営するサーバーに個人情報を送信して保存しません。**  
次に挙げるデータは、お使いのブラウザ環境内の **Chrome のローカルストレージ（`chrome.storage.local`）** に保存されます。

| データの種類 | 内容 | 用途 |
|-------------|------|------|
| プロフィール | 氏名、連絡先、住所、学歴など、設定画面で入力した項目 | フォームへの自動入力 |
| アクティブプロフィール ID | 現在選択中のプロフィール | どのプロフィールで入力するかの識別 |
| 設定 | ハイライト表示、プレビュー、入力遅延、フローティングボタンの表示など | 動作のカスタマイズ |
| 最近のマイページ一覧（訪問履歴） | 条件に合致したページの **URL**、**ページタイトル**、**訪問日時**（最大約100件） | ポップアップでのショートカット表示・バッジ表示の補助 |

訪問履歴は、タブの読み込み完了時に URL が一定のパターン（就活・採用関連サイトとみなすホスト名・パスのヒューリスティック）に合致した場合にのみ更新されます。すべての Web 閲覧履歴を記録するものではありません。

## 3. 第三者への送信（ネットワーク通信）

端末外へ送信が発生するのは、次の場合に限られます。

### 3.1 郵便番号検索 API（zipcloud）

住所補完のため、**7桁の郵便番号**が **株式会社インフォメーションセンターが提供する ZipCloud** に送信されます。

- エンドポイント例: `https://zipcloud.ibsnet.co.jp/api/search`
- 送信されるのは郵便番号に基づく検索リクエストです。

ZipCloud の取り扱いについては、同サービスの規約・ポリシーをご確認ください。

### 3.2 ファビコン画像（Google）

ポップアップの「最近のマイページ」一覧でサイトのアイコンを表示するため、**サイトのドメイン名**がクエリに含まれたリクエストとして **Google のファビコン取得サービス** に送られることがあります。

- 例: `https://www.google.com/s2/favicons?domain=…`

Google のデータの取り扱いについては、Google のプライバシーポリシーをご確認ください。

### 3.3 閲覧している Web ページ

コンテンツスクリプトにより、**ユーザーが開いたページの DOM 上で**フィールド検出・入力・ UI 表示が行われます。これらの処理で取得したフォームの内容を開発者サーバーに送信する機能はありません。

## 4. Chrome で要求する権限と理由

| 権限 | 理由 |
|------|------|
| `storage` | プロフィール・設定・訪問一覧の保存 |
| `tabs` | タブの URL 判定によるバッジ表示と訪問一覧の更新、アクティブタブの参照 |
| `activeTab` | ユーザーが操作中のタブへの処理に関連する機能 |
| `scripting` | コンテンツスクリプトが未注入のページへのフォールバック注入 |
| `host_permissions`（`<all_urls>`） | さまざまな企業の採用・マイページ上のフォームへ自動入力するため |

## 5. ユーザーの管理・削除

- 設定画面からプロフィールの編集・削除、データのエクスポート／インポートが可能です。
- ポップアップから「最近のマイページ」の一覧をクリアできます。
- 拡張機能をアンインストールすると、当該ブラウザプロファイル内の本拡張機能に紐づくローカルデータは通常削除されます（Chrome の仕様に依存します）。

## 6. お子様のプライバシー

本拡張機能は、13歳未満の方を対象としたサービスではありません。

## 7. ポリシーの変更

機能追加や法令・ガイドラインの変更に応じて、本ポリシーを更新することがあります。更新後はリポジトリ上の本ファイルの更新日を改めます。

## 8. 免責事項

本拡張機能は入力支援ツールです。自動入力後の内容は必ずご自身で確認し、送信の可否はご自身の責任で判断してください。
